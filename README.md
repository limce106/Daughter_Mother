# 엄마와 딸(Daughter-Mother)

2021년 겨울 서울여대 구루1 유니티분반 6조 [엄마와 딸] 팀입니다.

1학년 2학기 겨울방학 교내 프로그램 GURU1 유니티 분반에서 처음 제작한 게임이자 우수상을 수상한 게임입니다.

장르: 도트 그래픽 탑다운뷰 어드밴처 RPG

주제 : 잃어버린 엄마를 찾기 위함 모험, 어린 시절 엄마와 함께했던 그리운 추억 여행 

제작기간 : 2022.01.20 ~ 2022.02.07 (19일)

발표 링크 : <https://www.youtube.com/watch?v=ApTbvGVLYOA&t=1s>

본인 담당 파트:

게임 내 아이템과 장비 구상

카메라가 플레이어를 따라 움직이도록 구현

플레이어 움직임, 공격, 피격 구현

플레이어 피격 시 배경색 변화(빨간색으로 화면이 깜빡임) 구현

적(곰 인형, 달고나, 토끼 인형) 움직임, 공격, 피격 구현

적(곰 인형, 달고나, 토끼 인형) Ai 구현

적(곰 인형, 달고나, 토끼 인형) 공격 패턴 구현

플레이어 애니메이션 구현

플레이어가 장비를 얻으면 마법봉을 지닌 모습으로 변하도록 애니메이션 구현

가방 안에서 버둥거리던 토끼 인형의 모습이 대화 내용에 알맞게 가방에서 꺼내진 모습으로 바뀌도록 애니메이션 구현

적(곰 인형, 달고나, 토끼 인형) 애니메이션 구현

적은 플레이어가 말 걸기 전까지 멈춰있도록 구현

적과 플레이어의 대화가 끝난 후 적의 움직임, 공격, 피격이 시작하도록 구현

플레이어가 적을 쓰러뜨리면 적은 다시 멈추도록 구현

맵에서 적을 쓰러뜨리기 전까지 맵을 나가지 못하도록 맵 제한 구현

플레이어가 적을 쓰러뜨린 후 맵을 나갈 수 있도록, 맵 제한이 풀리도록 구현

적 쓰러뜨린 후 기억 장면(일러스트)을 띄웠다가 사라지도록 구현

대화 데이터 베이스 구현
